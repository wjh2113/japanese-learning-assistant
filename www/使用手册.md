# 日语学习助手使用手册

## 一、本地访问

### 1.1 启动本地服务器

打开终端，进入项目 `www` 目录：

```bash
cd D:/VSworkspace/CCtest/www
python -m http.server 8080 --bind 127.0.0.1
```

### 1.2 浏览器打开

访问：

```
http://127.0.0.1:8080
```

> 注意：不要直接用 `file://` 协议双击打开 `index.html`，否则 JavaScript 模块会无法加载。

---

## 二、首次配置

首次打开页面时，会自动弹出「设置」窗口。你需要填写以下信息：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| API Base URL | `https://api.deepseek.com/v1` | DeepSeek 官方 API 地址 |
| API Key | 空 | 你的 DeepSeek API Key |
| 默认模型 | `deepseek-v4-pro` | 当前使用的 AI 模型 |
| 默认等级 | N5 | JLPT 日语等级 |

### 2.1 获取 DeepSeek API Key

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 注册/登录账号
3. 进入「API Keys」页面，创建新的 API Key
4. 复制 Key 粘贴到设置中

### 2.2 测试连接

填写完成后，点击「测试连接」按钮。如果显示「连接成功」，即可正常使用。

如果显示「连接失败」，请检查：
- API Key 是否正确复制，不要有多余空格
- 网络是否可以访问 `api.deepseek.com`
- 账户余额是否充足

---

## 三、功能说明

### 3.1 听力练习

1. 选择 JLPT 等级和主题（或留空随机）。
2. 点击「生成听力材料」，AI 会生成一段日语听力内容。
3. 点击「▶️ 播放」听取日语朗读。
4. 可调节语速（0.5x - 1.5x）。
5. 可点击「隐藏原文」进行盲听训练。
6. 在听写框中输入听到的内容，点击「提交听写」查看相似度评分。

### 3.2 口语练习

1. 选择 JLPT 等级和场景。
2. 点击「生成口语题」，AI 会给出一句日语和中文提示。
3. 点击「🔊 播放标准发音」先听一遍。
4. 点击「🎙️ 开始录音」，用麦克风朗读日语。
5. 录音结束后，点击「让 AI 评估」获取发音/语法/自然度评分和改进建议。

> 口语练习需要浏览器支持 Web Speech API，建议使用 Chrome、Edge 或 Safari。

### 3.3 阅读练习

1. 选择 JLPT 等级和体裁。
2. 点击「生成阅读材料」，AI 会生成一篇日语文章。
3. 点击文章中的日语单词，可以查看读音和中文释义。
4. 遇到生词可点击「收藏单词」。
5. 阅读完成后，完成下方的理解测验，提交后查看得分和解析。

### 3.4 写作练习

1. 选择 JLPT 等级和题目（或留空随机）。
2. 在文本框中用日语写作。
3. 点击「提交批改」，AI 会给出综合分数、错误分析和修改后范文。
4. 可点击「🔊 朗读范文」听取修改后的文章。

## 四、学习记录

- 每次生成听力、口语、阅读、写作材料，都会自动保存到本地学习历史。
- 阅读中收藏的单词会保存到「收藏单词」列表。
- 数据保存在浏览器 `localStorage` 中，刷新页面不会丢失。
- 清除浏览器缓存或更换浏览器会导致数据丢失。

## 五、常见问题

### Q1：页面打开后空白，点击没有反应？

A：请确保：
1. 使用 `http://127.0.0.1:8080` 访问，不是 `file://`。
2. 按 F12 打开浏览器控制台，查看是否有红色报错。
3. 刷新页面重试。

### Q2：听不到日语朗读？

A：
1. 检查电脑/手机是否已安装日语语音包。
2. 检查浏览器是否静音。
3. 在 Chrome/Edge 中，Web Speech API 效果更好。

### Q3：语音录音失败？

A：
1. 确保浏览器有麦克风权限。
2. 检查麦克风是否正常工作。
3. 建议使用 Chrome 浏览器。

### Q4：AI 返回报错？

A：
1. 检查 API Key 是否正确。
2. 检查网络连接。
3. 查看设置中的 API Base URL 是否为 `https://api.deepseek.com/v1`。
4. 如果 DeepSeek 服务暂时不可用，稍后再试。

### Q5：如何切换模型？

A：点击右上角「设置」，修改「默认模型」即可。当前默认使用 `deepseek-v4-pro`。
